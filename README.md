## 🌟 What is AI Text Summarizer Pro?

AI Text Summarizer Pro is an intelligent NLP-powered platform that transforms **long-form content into concise, meaningful summaries**.

Whether it's:
- 📰 News articles  
- 📄 Research papers  
- 📚 Blogs  
- 📊 Reports  
- 📂 PDFs / DOCX / TXT files  

👉 This tool helps you **understand information faster, smarter, and efficiently.**

## 🎯 Key Features

### ✍️ Multi-Input Support
- Paste text directly
- Extract content from URLs
- Upload PDF, DOCX, TXT files

### 🧠 AI-Powered Summarization
- Uses state-of-the-art transformer models
- Generates meaningful, context-aware summaries

### 🎛️ Flexible Output Modes
- Short / Medium / Detailed summaries
- Paragraph or bullet-point format

### 🌐 Translation Support
- Convert summaries into multiple languages
- Expand usability globally

### 🔑 Keyword Extraction
- Highlights important terms automatically
- Helps quick understanding of core topics

### 📥 Export & Copy
- Download summaries
- Copy instantly for reuse

## 🧠 How It Works

```mermaid
graph TD
A[User Input] --> B[Text Extraction]
B --> C[Text Cleaning]
C --> D[Chunking Engine]
D --> E[Transformer Model]
E --> F[Summary Generated]

G --> H[Display & Download]

| Category          | Tools                     |
| ----------------- | ------------------------- |
| 🧠 AI/NLP         | Hugging Face Transformers |
| 🔥 Backend        | Python                    |
| 🎨 UI             | Streamlit                 |
| ⚙️ ML Framework   | PyTorch                   |
| 🌐 Web Extraction | Trafilatura               |

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt
streamlit run app.py
